FAQ

What is Aisis?

Aisis is a theme framework designed for WordPress to give developers tools to easily create child themes using a fully functional, easy to use API that gets out of your way and lets you do what you want to do with little or no limitations.

How is Aisis different from other theme frameworks?

Aisis is software. It is a set of API’s that allow the user to quickly and easily build a child theme with just a couple clicks of the mouse and a few strokes on the keyboard. Aisis provides all the tools you need a s a developer to build rich, engaging and powerful child themes.

Can Aisis be used a theme?

It can be and it provides you with some basic options for things like changing default text or editing the css or php files in the custom folder. However while Aisis provides you with these abilities its true power is in its ability to manipulated through its extensible API that includes things such as hooks and pluggable functions to name a few.

How easy is it to have the API in front of you when you develop?

It’s as easy as eating ice crème. Include the parent themes functions.php in your child themes functions.php and BOOM! Your dreams become soaked with all the functions, methods, classes and core API that is Aisis.
