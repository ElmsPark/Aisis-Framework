      This class is designed to allow to you to pass in the class name and its dependencies to then allow you to return a object of that class.

      There is a data structure that allows you to pass in the classes despendencies.

      $dependencies = array(
    'class_name' => array(
        'params' => array(
           'class_param',
           'class_param2'
        )
    ),
);

      Then all you need to do:

      $object = AisisCore_Factory_Pattern::create('class_name')
// From there you can do:
$object->class_name_method();
