      • \CoreTheme_Templates_View_Helpers_CustomLoop

      This class deatils and creates custom loops for Aisis.

      We extend the CoreTheme Loop which in turns extends the Aisis Core Loop, and then we build the various loops bellow. The way to use this class is to cll the custom_loop() function which will then build the loops for the front page. The other public function is the custom_post_types_loop which takes in either carousel or mini as the option and renders a loop based off of that.

      loop() 
      Inherited

      If we have options, then we check if were a single. If not do we have a query? if not then we just do the general loop.

      If none of this is true, we check the single.

      If we are not on the single post and the options are not set then we we will just call the general loop.

      If the single page is rendered, we also take care of loading the comments.php file as well.
