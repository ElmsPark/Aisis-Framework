AisisCore_Theme¶

This class is responsible for setting up the theme and it's core componenets.

<p>We take in an array that looks like:</p> <p><code> $options = array( 'widgets' => array(), 'navigation' => array(), 'theme_support' => array( 'post_formats' = array(), 'thumbnails' => array( 'exceptions' => array(), ) ), 'custom_theme' => array( 'custom-header' => array(), 'custom-background' => array() ) ); </code></p> <p>This then gets called into the class such as:</p> <p><code> $theme = new AisisCore_Theme($options); </code></p> <p>At any point you may call one of the public functions below to change the options, making sure to set the appopriate $key, $value in the $options array.</p>
